# Memory Controller

## Objective

To design and verify a Memory Controller using Verilog HDL.

## Features

- Read Operation
- Write Operation
- Address Decoding
- FSM-Based Control
- Memory Access Management

## Files

- memory_controller.v
- tb_memory_controller.v
- memory_waveform.png

## Tools Used

- Verilog HDL
- Icarus Verilog
- EPWave
- EDA Playground

## Verification

The Memory Controller was verified through simulation by performing read and write operations and validating data transfer between memory locations.

## Learning Outcomes

- Memory Interface Design
- FSM Design
- RTL Design
- Testbench Development
- Waveform Analysis

# UART Communication System

## Objective
To design and verify a complete UART communication system using Verilog HDL.

## Features
- UART Transmitter
- UART Receiver
- Baud Rate Generation
- Clock Generation
- Monitor Signals
- Serial Data Communication
- FSM-Based Control

## Files
- uart_communication_system.v
- tb_uart_communication_system.v
- uart_waveform.png

## Tools Used
- Verilog HDL
- Icarus Verilog
- EPWave
- EDA Playground

## Verification
The UART communication system was verified through simulation by transmitting and receiving serial data while monitoring baud rate timing, clock generation and state transitions.

## Learning Outcomes
- UART Protocol
- RTL Design
- FSM Design
- Testbench Development
- Waveform Analysis

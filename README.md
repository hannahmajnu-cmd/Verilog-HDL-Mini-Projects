# Verilog HDL Mini Projects

A collection of Verilog HDL projects developed and verified using simulation tools such as Icarus Verilog and EPWave.

## Projects Included

### 1. ALU Design
- 3-bit Arithmetic Logic Unit
- Arithmetic and Logical Operations
- Verilog HDL Implementation
- Testbench Verification

### 2. UART Communication System
- UART Transmitter and Receiver
- Baud Rate Generation
- Serial Communication
- FSM-Based Design

### 3. RISC Processor
- Program Counter
- ALU Operations
- Instruction Execution
- Register-Based Architecture

### 4. Memory Controller
- Read and Write Operations
- Address Management
- FSM Control Logic
- Memory Access Verification

## Tools Used
- Verilog HDL
- Icarus Verilog
- EPWave
- EDA Playground

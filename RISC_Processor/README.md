# RISC Processor

## Objective

To design and verify a simple RISC processor using Verilog HDL.

## Features

- Program Counter (PC)
- Instruction Fetch
- Arithmetic Logic Unit (ALU)
- Register Operations
- Control Logic
- Sequential Instruction Execution

## Files

- risc_processor.v
- tb_risc_processor.v
- risc_waveform.png

## Tools Used

- Verilog HDL
- Icarus Verilog
- EPWave
- EDA Playground

## Verification

The processor was verified through simulation by executing instructions and observing Program Counter updates and register operations.

## Learning Outcomes

- Processor Architecture
- RTL Design
- Datapath Design
- Control Unit Design
- Testbench Development
- Waveform Analysis

# 3-bit ALU Design

## Objective
To design and verify a 3-bit Arithmetic Logic Unit using Verilog HDL.

## Features
- Addition
- Subtraction
- AND
- OR
- XOR

## Files
- alu.v
- tb_alu.v

## Tools Used
- Verilog HDL
- Icarus Verilog
- EPWave

## Verification
The ALU was verified using a testbench and waveform analysis.
